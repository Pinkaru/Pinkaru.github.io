# Pinkaru's Technical Blog

이 블로그는 기술 블로그로 개편되었습니다. 아래 기능들을 활용하여 글, 사진, 포트폴리오를 관리할 수 있습니다.

## 새로운 기능

### 1. 카테고리 (Categories)
- 기술별로 포스트를 분류할 수 있습니다
- 네비게이션 메뉴에서 "Categories" 클릭하여 모든 카테고리 확인
- 사이드바에서 주요 카테고리 바로가기

### 2. 태그 (Tags)
- 키워드로 포스트를 분류할 수 있습니다
- 네비게이션 메뉴에서 "Tags" 클릭하여 모든 태그 확인
- 사이드바에서 인기 태그 바로가기

### 3. 포트폴리오 (Portfolio)
- 프로젝트와 작업물을 전시할 수 있습니다
- 네비게이션 메뉴에서 "Portfolio" 클릭

## 포스트 작성 방법

### 일반 블로그 포스트
```markdown
---
layout: post
title: "포스트 제목"
date: 2024-03-01
tags: [Python, Django, Web Development]
categories: Web Development
---

포스트 내용...
```

### 포트폴리오 아이템
```markdown
---
layout: post
title: "프로젝트 이름"
date: 2024-03-01
tags: [React, Node.js]
categories: Portfolio
---

프로젝트 설명...
```

## 이미지 업로드
이미지는 `/assets/images/` 폴더에 저장하고 다음과 같이 사용:
```markdown
![이미지 설명](/assets/images/your-image.jpg)
```

## 검색 기능
- 상단 검색바를 통해 Google 사이트 검색 지원
- 카테고리와 태그를 통한 필터링 가능
